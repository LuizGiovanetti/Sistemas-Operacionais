# Trabalho 1 - Sistemas Operacionais: Windows Vista

## Sobre o Projeto

Este repositório contém o material desenvolvido para a disciplina de **Sistemas Operacionais**, tendo como tema de estudo o **Windows Vista**.

O trabalho foi elaborado em dupla com o objetivo de apresentar aspectos históricos, características técnicas e uma demonstração prática do sistema operacional utilizando técnicas de virtualização. A atividade seguiu os critérios definidos pelo professor, abordando desde o contexto de criação do sistema até seus principais recursos e funcionalidades.

## Objetivos

* Apresentar o contexto histórico do Windows Vista;
* Identificar suas principais características técnicas;
* Analisar recursos relacionados a gerenciamento de processos, memória e segurança;
* Demonstrar o funcionamento do sistema operacional em ambiente virtualizado;
* Desenvolver habilidades de pesquisa, síntese e apresentação de conteúdo técnico.

## Conteúdo Desenvolvido

O trabalho aborda os seguintes tópicos:

* Introdução aos Sistemas Operacionais;
* Histórico e contexto de lançamento do Windows Vista;
* Características de multiusuário e multitarefa;
* Gerenciamento de processos e recursos;
* Arquitetura 32 e 64 bits;
* Suporte a múltiplos núcleos (multicore);
* Sistema de arquivos NTFS;
* Recursos de segurança e proteção, incluindo UAC (User Account Control);
* Demonstração prática utilizando virtualização.

## Ferramentas Utilizadas

- **Canva**: desenvolvimento dos slides e organização visual da apresentação;
- **Máquina Virtual**: execução e demonstração prática do Windows Vista em ambiente virtualizado;
- **Windows Vista**: sistema operacional estudado e analisado durante o trabalho.
